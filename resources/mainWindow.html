Application Window

☰

The application consists of one main window divided into three panes. The left pane is the Trip Data pane and shows the parsed contents of all trips in the EarthTrack log file, the right pane is the Trip Summary pane and shows summary information for the currently selected trip, and the bottom pane shows a speed plot for the currently selected trip.

There is a blue status bar at the bottom of the application window that displays temporary messages, as well as showing the current epoch trip times are displayed in, and the controller ID of the current trip (empty when no trip displayed).
