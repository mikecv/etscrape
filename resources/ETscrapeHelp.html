EarthTrack Log Scraper Help

☰

The EarthTrack Log Scraper is relatively simple in that it just reads the contents of a debug log file downloaded from an EarthTrack Fleet Controller module or message log file downloaded from the EarthTrack server, and displays a sanitised version of trips included in the log file.

The primary purpose of the application is as a diagnostic tool for test personnel; so that they can avoid having to look through the log file which is not as easy to read.

Use the side menu to browse to specific help topics.
